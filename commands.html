Server Commands

🎛️ Free StreamDeck Profile available — requires an Elgato Stream Deck

Note: If some commands doesn't work or you have any issues, please let us know on discord to John Mack.

Essential Commands

/spawn

Teleport to the server spawn point

/home [name]

Teleport to your home location

/sethome [name]

Set a home at your current location

/delhome [name]

Delete a home location

/rtp

Random teleport to a safe location

/kit

List all available kits

/jobs

Open the jobs menu

/market

Open the player market

/skills

Open skill attributes GUI to allocate points

/skills augments

View and manage your augments

/eg Bestiary

Open the Bestiary menu

/eg Bounty

Open the Bounty menu

Communication Commands

/msg [player] [message]

Send a private message to a player

/r [message]

Reply to the last private message

/mail

Open your mailbox

/mail send [player] [message]

Send mail to another player

/ignore [player]

Ignore a player's messages

/list

Show all online players

Teleport Requests (TPA)

/tpa [player]

Request to teleport to a player

/tpahere [player]

Request a player to teleport to you

/tpaaccept

Accept a teleport request

/tpadeny

Deny a teleport request

Land Claims (SimpleClaims)

/sc

Open the claiming party GUI

/sc claim

Claim the chunk you're standing on

/scp

Open claim settings interface

Economy Commands

/bank

Access your bank (EcotaleCoins)

/bank deposit all

Deposit all coins into your bank

/bank withdraw all

Withdraw all coins from your bank

Utility Commands

/rules

Display server rules

/discord

Get Discord invite link

/motd

Show message of the day

/afk

Toggle AFK status

/playtime

Check your playtime

/vote

Check the vote list

/voile

Open the Documentation menu with plenty of useful information about some of our Mods and the Server

/shownameplate

Toggle your nameplate visibility in third-person

/hyclockmenu

Open the Clock configuration menu
